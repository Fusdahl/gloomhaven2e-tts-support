Drop your 3 final Silent Knife extra cards here.

Required naming (recommended):
- 01_silent_knife_perk_1.png
- 02_silent_knife_perk_2.png
- 03_silent_knife_extra_stock.png

Notes:
- Any image format is okay: png/jpg/jpeg/webp.
- Build script will normalize each to 814x530 (Quartermaster tile size).
- Files prefixed with REFERENCE_ are ignored by the builder.

After dropping files, run:
python3 src/build_silent_knife_extra_perk_atlas.py

- assets/final_class_amd_atlases/silent_knife_extra_perks_atlas_2x2.png
