# Silent Knife AMD Transformation Input Pack (v2)

Use each numbered folder as a single, self-contained img2img task.

## General use
1. Open one folder.
2. Read `TASK.txt`.
3. Attach `BASE_*` + `DONOR_*` + `REF_*` images from that folder.
4. Paste `PROMPT.txt` into your img2img tool.
5. Save output using `OUTPUT_FILENAME.txt` (if present).

## 01_disarm_plus1
Goal: create a `+1 disarm` face.

## 02_money_token_plus1
Goal: create a `+1` money-token conditional face.

## 03_invisible_plus1_self
Goal: convert `137816` into green style and update to `+1` while preserving intended effect semantics.

## 04_reuse_no_edit
No generation; these are direct-reuse cards.
