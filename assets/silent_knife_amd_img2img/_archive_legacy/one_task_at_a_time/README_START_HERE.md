# One Task At A Time Queue (Silent Knife AMD)

Process only one folder at a time in numeric order.

## How to run each transform task
1. Open `NN_task_name/`.
2. Attach `INPUT_reference_card.png` and `INPUT_perks_panel_zoom.png` to your img2img model.
3. Paste `PROMPT.txt`.
4. Ensure target effect in `TASK.txt` matches your intended icon/text.
5. Save result using exact name in `OUTPUT_FILENAME.txt`.
6. Put generated output into `done_drop_here/`.

## Keep tasks
- Tasks 09-11 are keep-as-is copy tasks (no AI generation).

## Notes
- Task 04 is marked optional as an extra money-token variant.
- This queue is intentionally linear and explicit to reduce ambiguity.
